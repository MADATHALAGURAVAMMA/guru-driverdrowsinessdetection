# guru-driverdrowsinessdetection
# 🚗 Driver Drowsiness Detection System

This is a Flask-based web application that detects driver drowsiness using computer vision and alerts the user using a sound alarm.

---

## 📌 Features
- User Registration & Login system
- Live webcam streaming
- Eye blink detection (EAR method)
- Yawning detection (MAR method)
- Alarm sound alert when drowsy
- Simple web interface using Flask

---

## 🛠️ Technologies Used
- Python
- Flask
- OpenCV
- MediaPipe
- NumPy
- SciPy
- TensorFlow (optional CNN model)
- Pygame
